## INTRODUCTION

Created in 1970 and design by Donald Chamberlin and Raymond Boyce.

« _Structured Query Language (SQL) is a domain-specific language used to manage data, especially in a relational database management system (RDBMS). It is particularly useful in handling structured data. The scope of SQL includes data query, data manipulation (insert, update, and delete), data definition (schema creation and modification), and data access control._ » — [Wikipedia](https://en.wikipedia.org/wiki/SQL)
