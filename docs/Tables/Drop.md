| Engine | Code |
|-:|:-|
| PostgreSQL | `DROP TABLE {{table_name}};` |
| MariaDB | `DROP TABLE {{table_name}};` |
| SQL Server | `DROP TABLE {{table_name}};` |

- Replace `{{table_name}}` with the name of the table.
