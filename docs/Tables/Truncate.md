# Truncate

Clear all data from the table without deleting it.

| Engine | Code |
|-:|:-|
| PostgreSQL | `TRUNCATE TABLE {{table_name}};` |
| MariaDB | `TRUNCATE TABLE {{table_name}};` |
| SQL Server | `TRUNCATE TABLE {{table_name}};` |

- Replace `{{table_name}}` with the name of the table.
