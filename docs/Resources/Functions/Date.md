# Date

## Now

**MariaDB & PostgreSQL**
```SQL 
NOW()
```

**SQL Server**
```SQL
GETDATE()
```

## Year, month, minute...
**MariaDB & SQL Server**
```SQL
YEAR(value)
MONTH(value)
MINUTE(value)
```

**PostgreSQL**
```SQL
EXTRACT(YEAR FROM value)
EXTRACT(MONTH FROM value)
EXTRACT(MINUTE FROM value)
```
