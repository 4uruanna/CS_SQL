## Arithmetic

| Operator | Description |
|-:|:-|
| `+` | Add |
| `-` | Substract |
| `*` | Multiply |
| `/` | Divide |
| `%` | Modulo (Remainder of the division) |
| `&` | Bitwise AND |
| `\|` | Bitwise OR |
| `^` | Bitwise XOR |
| `~` | Bitwise NOT |

## Examples

```SQL
SELECT 1 + column_a
FROM table_name;
```

```SQL
UPDATE table_name
SET column_a = column_a - 1
WHERE {{CONDITION}};
```
