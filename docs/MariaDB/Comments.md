# Comments

Comments are used to explain or prevent execution code:
- `/* */` allows you to comment on one or more lines.
- `--` allows you to comment on a line.

> Note: A comment has no effect on the query.

```SQL
-- This is a single line comment 
```

```SQL
/*
    That is a
    multi-line
    comment.
*/ 
```
