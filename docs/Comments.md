# Comments

A comment has no real effect on the query.

`--` allows you to comment on a line.
```SQL
-- This is a single line comment 
```

`/* */` allows you to comment on one or more lines.
```SQL
/*
    That is a
    multi-line
    comment.
*/ 
```
