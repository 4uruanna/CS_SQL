# Create

_Create new database._

| Engine | Code |
|-:|:-|
| PostgreSQL | `CREATE DATABASE {{database_name}};`|
| MariaDB | `CREATE DATABASE {{database_name}};`|
| SQL Server | `CREATE DATABASE {{database_name}};`|

Replace `{{database_name}}` with the name of your choice.
