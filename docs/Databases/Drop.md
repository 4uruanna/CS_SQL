# Drop

_Delete database._

| Engine | Code |
|-:|:-|
| PostgreSQL | `DROP DATABASE {{database_name}};`|
| MariaDB | `DROP DATABASE {{database_name}};`|
| SQL Server | `DROP DATABASE {{database_name}};`|

Replace `{{database_name}}` with the name of the target database.
